En interaktiv tic-tac-toe spel komponent där användare kan konkurrera mot en datoropponent. Komponenten följer Midas designriktlinjer med fullständig tillgänglighetsstöd och tangentbordsnavigation.

```tsx
import { TicTacToe } from '@midas-ds/components'
```

```tsx
<TicTacToe />
```

<div className='card'>
  <TicTacToe />
</div>

## Funktioner

- Tre svårighetsgrader (easy, medium, hard)
- Dator AI-motståndare med minimax-algoritm för svår svårighetsgrad
- Fullständigt tillgänglighetsstöd med ARIA-etiketter och skärmläsarmeddelanden
- Stöd för tangentbordsnavigation
- Responsiv design
- Anpassningsbara etiketter och spelarsymboler

## Användning

### Grundläggande spel

Komponenten skapar automatiskt ett 3x3 rutnät där användaren kan klicka för att placera sin symbol. Datorn kommer att svara automatiskt efter användarens drag.

### Tangentbordsnavigation

- Använd **Tab** för att navigera mellan celler och knappar
- Använd **Enter** eller **Space** för att välja en cell eller aktivera en knapp
- Spelbrädet meddelar aktuell status till skärmläsare

### Tillgänglighet

Komponenten inkluderar:
- ARIA-etiketter för alla interaktiva element
- Statusuppdateringar som meddelas till skärmläsare
- Fokushantering för tangentbordsanvändare
- Tydlig visuell feedback för spelstatus

## Riktlinjer

### När ska komponenten användas

- I utbildningsapplikationer för att lära ut spelteori eller strategi
- Som pausaktivitet i längre formulär eller arbetsflöden
- I spelsamlingar eller nöjesapplikationer
- För att demonstrera interaktiva komponenter i designsystem

### Anpassning

- Använd `difficulty` för att anpassa utmaningsnivån baserat på målgrupp
- Anpassa `resetButtonLabel` och `newGameButtonLabel` för lokalisering
- Använd `onGameStateChange` för att integrera med analytics eller spelprogression
