Rundad knappkomponent som visar en gul glöd när den trycks ned. Perfekt för flytande åtgärdsknappar eller ikon-endast interaktioner.

```tsx
import { RoundButton } from '@midas-ds/components'
import { Plus } from 'lucide-react'
```

```tsx
<RoundButton
  icon={Plus}
  aria-label="Add item"
/>
```

<div className='card'>
  <RoundButton
    icon={Plus}
    aria-label="Add item"
  />
</div>

## Varianter

### Primär

Primär rundknapp används för den viktigaste handlingen, som att lägga till nytt innehåll eller bekräfta en åtgärd.

```tsx
<RoundButton
  icon={Plus}
  aria-label="Add item"
/>
```

<div className='card'>
  <RoundButton
    icon={Plus}
    aria-label="Add item"
  />
</div>

### Sekundär

Sekundär rundknapp används för underordnade handlingar som inställningar eller konfiguration.

```tsx
<RoundButton
  variant="secondary"
  icon={Settings}
  aria-label="Settings"
/>
```

<div className='card'>
  <RoundButton
    variant="secondary"
    icon={Settings}
    aria-label="Settings"
  />
</div>

### Tertiär

Tertiär rundknapp används för mjuka handlingar som att gilla eller markera innehåll.

```tsx
<RoundButton
  variant="tertiary"
  icon={Heart}
  aria-label="Like"
/>
```

<div className='card'>
  <RoundButton
    variant="tertiary"
    icon={Heart}
    aria-label="Like"
  />
</div>

### Fara

Farlig rundknapp används för destruktiva handlingar som att ta bort eller stänga.

```tsx
<RoundButton
  variant="danger"
  icon={X}
  aria-label="Close"
/>
```

<div className='card'>
  <RoundButton
    variant="danger"

RoundButton kan också innehålla text istället för eller tillsammans med en ikon.


En inaktiverad rundknapp kan inte interageras med.

```tsx
  icon={Plus}
  aria-label="Add item"
  isDisabled
```

## Riktlinjer

### Val av komponent

- Använd RoundButton för flytande åtgärdsknappar (FAB - Floating Action Button)
- Perfekt för ikon-endast interaktioner där den runda formen betonar åtgärden
- Använd för snabba åtgärder som "lägg till", "favorit", "stäng" eller "inställningar"

### Användningsområden

- Flytande åtgärdsknappar som följer med användaren vid skrollning
- Kompakta gränssnitt där utrymmet är begränsat
- Ikon-baserade navigeringselement
- Snabba åtgärder i listor eller kort

### Tillgänglighet

- Använd alltid `aria-label` för att beskriva knappens funktion
- Se till att ikonen är tydlig och förståelig för användaren
- Testa med skärmläsare för att säkerställa god tillgänglighet

### Gul glöd-effekt

RoundButton har en unik gul glöd-effekt när den trycks ned, vilket ger användaren tydlig visuell feedback. Denna effekt är automatisk och kräver ingen extra konfiguration.
